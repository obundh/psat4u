# PSAT4U Privacy Policy

Last updated: May 22, 2026

PSAT4U is a free, unofficial, offline-first study app for practicing Korean Public Service Aptitude Test style skills.

## Data Collection

PSAT4U does not collect, transmit, sell, or share personal data.

The app does not use accounts, remote servers, advertising SDKs, analytics SDKs, crash-reporting SDKs, push notifications, or remote API-backed question generation.

## Local Data Stored On Device

PSAT4U stores the following study data locally on the user's device:

- Training statistics
- Wrong-answer notes
- Timing records
- Difficulty and app preferences

This data stays on the device unless the user exports a backup file manually. It is not transmitted to the developer or to a third party.

## Data Deletion

Users can delete local study data from the app settings by using the reset function. Users can also uninstall the app to remove app-local data managed by Android.

## Third-Party Software

PSAT4U uses open source runtime software such as React, Capacitor, and Lucide. The current app build does not include third-party SDKs designed to collect personal or sensitive user data.

## Official Status

PSAT4U is not affiliated with, endorsed by, or sponsored by any exam administrator, public agency, trademark holder, publisher, or test-prep company.
