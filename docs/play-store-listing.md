# PSAT4U Play Store Listing Draft

## App Name

PSAT4U

## Short Description

오프라인으로 짧게 푸는 PSAT 유형별 틈새 훈련 앱

## Full Description

PSAT4U는 이동 중이나 쉬는 시간에 PSAT 감각을 유지하도록 만든 무료 오프라인 학습 앱입니다.

원하는 유형을 골라 오늘의 훈련을 만들고, 문제를 풀자마자 정답 여부와 풀이 시간을 확인할 수 있습니다. 유형별 기록과 오답노트를 기기 안에 저장해 다음 훈련에서 바로 다시 풀 수 있습니다.

주요 기능

- 오늘의 PSAT 훈련: 원하는 유형을 체크해서 짧은 훈련 코스 시작
- 난이도 설정: 초급, 중급, 고급 및 난이도 고정
- 유형별 기록: 풀이 시간, 정답률, 최고 기록 저장
- 오답노트: 틀린 문제를 저장하고 랜덤 재훈련
- 나에게 맞는 풀이 찾기: 유형별 여러 풀이법을 비교
- 오프라인 작동: 서버, 로그인, 광고, 분석 SDK 없이 사용

PSAT4U는 공직적격성평가 대비용 비공식 학습 도구입니다. 시험 주관기관, 상표권자, 출판사, 강의 업체와 제휴하거나 보증받은 앱이 아닙니다.

앱에 포함된 훈련 문제는 앱 내부 규칙으로 생성한 연습문제이며 공식 기출 원문, 상업 교재 문항, 유료 강의 자료를 번들하지 않습니다.
